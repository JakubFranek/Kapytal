Kapytal is a PyQt-based Windows tool for managing your personal or household finances. Log your income and expenses, track your investments, analyse the data and take control of your financial life.

## Features

- a wide variety of [Transactions](./docs/user/glossary.md#transaction): [Income](./docs/user/glossary.md#income-), [Expense](./docs/user/glossary.md#expense-), [Refund](./docs/user/glossary.md#expense-), [Cash Transfer](./docs/user/glossary.md#cash-transfer-), [Buy](./docs/user/glossary.md#buy-), [Sell](./docs/user/glossary.md#sell-), [Dividend](./docs/user/glossary.md#dividend-), [Security Transfer](./docs/user/glossary.md#security-transfer-)
- multi-[currency](./docs/user/glossary.md#currency-) support with [exchange rate](./docs/user/glossary.md#exchange-rate-) history
- [investment](./docs/user/glossary.md#security-) tracking and performance analysis
- multiple user support thanks to unlimited [tree-like](./docs/user/glossary.md#account-tree-) [Account](./docs/user/glossary.md#account) hierarchies
- advanced Transaction categorization with [Payees](./docs/user/glossary.md#payee-), [Tags](./docs/user/glossary.md#tag-) and hierarchical [Categories](./docs/user/glossary.md#category-)
- a variety of chart-based [Reports](./docs/user/glossary.md#reports) to analyse and visualize your data
- 14 powerful [Transaction filters](./docs/user/glossary.md#transaction-filters-) for precise Report data set control
- and much more...

## Requirements

- **Operating System:**
  - Windows 8 or newer
  - Linux with glibc 2.39+ (Ubuntu 24.04+, Debian 13+, Rocky Linux 9+)
- **Disk Space:** ~200 MB
- **Internet:** Optional (used for update check and exchange rate/security price updates only)

## Get started

- visit the [Quick Start Guide](/docs/user/quick_start_guide.md) for a video tutorial
- download the installer (Windows) or executable AppImage (Linux) from the [latest release](https://github.com/JakubFranek/Kapytal/releases/latest) assets
